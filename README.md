# Project description

In this project I have made the design for a PCB that will keep track of the position of a target that it is mounted to.

It is the prototype of a tracker that will update the position using a GPS sensor and a LoRa module for communication with a centralized server.

Up until this point the PCB will be used to test AT command upon arrival.
